# Phase 1 — Build the Lab Foundation

⬅️ [Back to Project Overview](../README.md)

## Introduction
This phase focuses on building a clean and properly segmented lab foundation.

## Network Design
- Proxmox bridges for WAN and multiple LANs
- pfSense as central firewall and router

## Segmentation Strategy
- Default deny between networks
- Explicit rules only where required

## Result
A stable and segmented environment ready for centralized logging and security testing.
